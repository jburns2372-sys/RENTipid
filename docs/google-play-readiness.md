# Google Play Store Readiness Checklist

## App Metadata
- [ ] App Name: RENTipid
- [ ] Short Description (80 chars max)
- [ ] Full Description (4000 chars max) - Must include safety features and verification details.
- [ ] High-res icon (512x512)
- [ ] Feature Graphic (1024x500)
- [ ] Phone Screenshots (Min 2, up to 8)
- [ ] Tablet Screenshots (7-inch and 10-inch)

## Privacy & Policy
- [ ] Privacy Policy URL active (`https://rentipid.com/privacy`)
- [ ] Data Safety Form completed (Declare Data types: Personal Info, Photos, Device IDs)
- [ ] Account Deletion URL active (`https://rentipid.com/account/delete`)

## Testing
- [ ] Internal Testing Track setup with at least 20 testers.
- [ ] Sandbox Payment flows verified working without crashing WebView.

## Release Management
- [ ] App Signing key generated and stored securely.
- [ ] Target SDK updated to latest Android requirement (SDK 34+).
