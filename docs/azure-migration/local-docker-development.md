# Local Docker Development\n\nTo safely develop and test the Azure backend locally without pushing to the cloud, use the following instructions:\n\n## 1. Build the API Container\nNavigate to the `apps/api` directory and run:\n```bash\ndocker build -t rentipid-api:local .\n```\n\n## 2. Run the Container locally\n```bash\ndocker run -p 3000:3000 --env-file .env rentipid-api:local\n```\n\n## 3. Verify Health Endpoints\nCheck that the container is responding to Azure-style health probes:\n- Liveness: `curl http://localhost:3000/health/live`\n- Readiness: `curl http://localhost:3000/health/ready`
