# PayMongo Live Approval & Follow-Up Checklist

To successfully transition from Phase 19B-B Pending into actual Live transactions, the RENTipid Super Admin must submit the following documents to PayMongo and track their approval.

## Required Documents Submitted
- [ ] Business registration submitted (e.g., Mayor's Permit)
- [ ] SEC / DTI documents submitted
- [ ] BIR documents submitted (if required, e.g., Form 2303)
- [ ] Authorized representative valid ID submitted
- [ ] Bank account proof submitted (e.g., Bank Statement / Passbook)

## Technical URLs Submitted
PayMongo requires these links to be live and fully accessible before they approve your account for live processing:
- [ ] Website/app URL submitted (`APP_BASE_URL`)
- [ ] Terms and Conditions URL submitted (`/terms`)
- [ ] Privacy Policy URL submitted (`/privacy`)
- [ ] Refund Policy URL submitted (can be within terms)
- [ ] Support/contact email submitted
- [ ] Product/service description submitted
