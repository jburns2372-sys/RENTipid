# Phase 3 Known Limitations and Phase 4 Deferred Items

## Known Limitations
- Phase 3 remains advisory only.
- Rules are initialized strictly as DRAFT.
- Rule activation remains separately controlled (manual intervention required to move from DRAFT to ACTIVE).
- No automated account restriction.
- No automated booking cancellation.
- No automated KYC rejection.
- No automated payment hold or release.
- No automated payout hold.
- No automatic emergency freeze.
- No listing-status mutation.
- No automated dispute or damage-claim mutation.
- No automated countermeasure.
- Production worker settings require final configuration before deployment.
- Final UAT is still required.
- Full production activation is not part of Gate 3H documentation work.

## Phase 4 Deferred Areas
- Incident case management.
- Response playbooks.
- Approval workflows.
- Dual control.
- Reversible response executors.
- Session revocation.
- Temporary account restrictions.
- Payment or payout holds.
- KYC manual-review escalation.
- Booking and listing review holds.
- Response simulation.
- Response rollback.
- Countermeasure governance.
- Phase 4 safety and acceptance testing.
