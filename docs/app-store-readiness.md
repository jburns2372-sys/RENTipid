# Apple App Store Readiness Checklist

## App Metadata
- [ ] App Name: RENTipid (max 30 chars)
- [ ] Subtitle (max 30 chars)
- [ ] Description (max 4000 chars)
- [ ] Promotional Text (max 170 chars)
- [ ] App Icon (1024x1024, no transparency)
- [ ] Screenshots (iPhone 6.5" and 5.5" minimum)
- [ ] App Preview Video (Optional but recommended)

## Privacy & Policy
- [ ] Privacy Policy URL active
- [ ] App Privacy Nutrition Label completed in App Store Connect
- [ ] **Critical:** Account Deletion flow verified and accessible without leaving the app.

## App Store Guidelines Adherence
- [ ] Guideline 3.1.1 (In-App Purchase): Ensure rental payments using external gateways (PayMongo) do not violate digital goods rules (Physical rentals are exempt, but must be clearly documented in review notes).
- [ ] Guideline 1.2 (User Generated Content): Ensure there is a way to flag/report offensive listings and block abusive users.
- [ ] Guideline 5.1.1 (Data Collection): Ensure camera/photo library requests have clear usage descriptions in `Info.plist`.

## Release Management
- [ ] TestFlight internal and external testing configured.
- [ ] Demo account credentials provided to App Store Review team.
